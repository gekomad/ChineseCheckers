### c:
`gcc -O3 main.c -o checkers`

`./checkers`


### rust:

`cd rust`

`cargo build --release`

`./target/release/checkers`

### vala:

`cd vala`

`valac ChineseCheckers.vala --Xcc=-O3 `

`./ChineseCheckers`

### scala:

`cd scala`

`sbt run`
